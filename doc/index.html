How to Build Engine2D

The Current Makefile is designed for OSX though with a little effort it should work on Linux as well.

Although this code was originally designed in mind for Windows it's been a little while since I've compiled it on Windows so right now it won't compile with rewriting some functions.

All of the Library's on OSX came from Homebrew. On Linux you should be able to use your distro's package manager

Librarys Needed:

  • Freetype 2 Version: 2.4.11
  • glew Version: 1.9.0
  • glfw Version: 2.7.7
  • physfs Version: 2.03
  • v8 Version: 3.18.2
  • freeimage Version: 3.15.4
  • sqlite3 Version: 3.7.16.2

sys - System Functions and Variables

sys.println or console.log (...)

Prints to the console, this may not be visible so I recommend writing to somewhere in the game window.

sys.runFile (string filename, boolean autoReload)

Loads filename. Unlike require() in node.js this acts more like appending a script tag in a browser so it exists in the same scope. Like require() in node.js this will automatically append the .js file extension

sys.drawFunc (function() drawFunc)

This method is where you should do all your drawing, there will be undefined behavior if you draw outside of this method.

sys.keyboardFunc (function(string type, string key, boolean press) keyboardFunc)

This method gets called each time you press or release a key. Type can be char/special depending on if the key was 'a' or 'ESC'. Press is true when the key was pressed and false when it's released.

sys.getGLVersion ( )

Returns a Object containing:

  • major: OpenGL Major Version
  • minor: OpenGL Minor Version
  • rev: OpenGL Revision

sys.microtime ( )

Returns a floating point number from gettimeofday. On UNIX platforms this should be microsecond accurate but Windows returns up to millisecond accuracy (this may become untrue as I improve the windows version).

sys.heapStats ( )

Returns a Object containing:

  • heapLimit: Heap Limit
  • heapTotalSize: Heap Total Size
  • heapTotalExecSize: Heap Total Executable Size
  • heapUsed: Heap Used

sys.screenWidth and sys.screenHeight

Width and Height of the current display area.

sys.platform

Returns one of "Windows", "Linux", "Darwin (OSX)" depending on what platform your on.

draw - Drawing Functions

draw.rect (number x, number y, number width, number height)

Draws a rectangle with the current color at x, y with a size of width, height.

draw.grid (number x, number y, number width, number height)

Draws a rectangle with the current color at x, y with a size of width, height. The difference if this method only draws the outline of the rectangle.

draw.grad (number x, number y, number width, number height, number color1, number color2, boolean vertical)

Draws a rectangle with the current color at x, y with a size of width, height. The rectangle is colored in a gradient between color1 and color2, both colors are to be specified in the form 0xrrggbb. vertical rotates the rectangle 90 degrees.

draw.draw (image image, number x, number y, number width, number height)

Draws image at x, y with a size of width, height

draw.setColorF (number r, number g, number b)

Sets the current drawing color to r, g, b. Each should be between 0 and 1

draw.setColor (number color)

Sets the current drawing color to color which should be specified as a hex number in the form 0xrrggbb.

draw.setColorI (number r, number g, number b)

Sets the current drawing color to r, g, b. Each should be between 0 and 255

draw.clearColor (number r, number g, number b)

Sets the current background color to r, g, b. Each should be between 0 and 1

draw.print (number x, number y, string text)

Draws text at x, y with the current color.

draw.openImage (string path)

Loads and returns a image from path. This method currently supports these formats

  • .tga: Truevision TGA

draw.cameraReset ( )

Reset the current camera view.

draw.cameraPan (number x, number y)

Pans the camera by X and Y. Camera Functions stack on top of each other and can be combined for useful effects.

draw.cameraZoom (number factor)

Zooms the camera by factor. Camera Functions stack on top of each other and can be combined for useful effects.

draw.cameraRotate (double rotation)

Rotates the camera by rotation. Camera Functions stack on top of each other and can be combined for useful effects.

draw.getTextWidth (string text)

Returns the width of the text specified.

draw.getVerts ( )

Returns the number of vertices's that have been drawn this frame.

draw.setCenter (number x, number y)

Offset's all drawing by x, y. Useful to rotate stuff around a point.

input - Current state of input devices

input.keyDown(string keycode)

Returns if keycode is currently down, keycode needs to be uppercase

input.mouseX and input.mouseY

Updated each frame with the current mouse location.

input.leftMouseButton and input.rightMouseButton

Updated each frame with the current state of the left and right mouse buttons.

fs - Restricted Filesystem Access

fs.readFile (string filename)

Returns the content of the file.

fs.writeFile (string filename, string content)

Writes content to filename. fs.configDir needs to be called before you can write a file.

fs.fileExists (string filename)

Returns if the file exists.

fs.fileSize (string filename)

Offset's all drawing by x, y. Useful to rotate stuff around a point.

fs.mountFile (number x, number y)

Offset's all drawing by x, y. Useful to rotate stuff around a point.

fs.configDir (number x, number y)

Offset's all drawing by x, y. Useful to rotate stuff around a point.

fs.mkdir (number x, number y)

Offset's all drawing by x, y. Useful to rotate stuff around a point.
